# packing-list

An app that lets you create a packing list for an upcoming trip. The app allows you to create multiple lists, set the date you are leaving, and the ability to reset all items on the list to not checked it reuse the list.

Currently working on:
* testing out using local storage for users without needing passwords.
* dark/light mode based on device preferences.
* Currently pausing development on this app to learn node.js to create an api for this app
