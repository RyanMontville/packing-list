# PackingList
----Currently in development----

An Angular app that creates packing lists for an upcoming trip. The features of the app include:
* Log in with a username, the app uses localStorgae to save username to device. If the username is found on the server, the app will display all lists created by the user
* Set the date you are leaveing for your trip
* Reset all items to unchecked and set a new leaving date without having to remake the list
* The app can use tts to read off instructions from a list
